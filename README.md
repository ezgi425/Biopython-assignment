# Biopython-assignment
group4 

## Description
This repository contains a Python script that parses protein FASTA files
downloaded from the NCBI Protein database, prints sequence identifiers, and
converts the files to GenBank format using Biopython.

## Protein Sources
- Hemoglobin subunit alpha (Accession: P69905)
- Hemoglobin subunit beta (Accession: NP_000509.1)
- Insulin (Accession: P01308)

Source: NCBI Protein database
